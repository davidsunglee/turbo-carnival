Implement **Galaxy 3: The Zenith Armada Grid** as a complete, implementation-ready stage that fits into the current game flow and can be expanded by the planner into concrete work items.

## Goal
Build Galaxy 3 end-to-end so the player can start a run from the beginning of the game, progress through Galaxies 1 and 2, defeat the Galaxy 2 boss, transition into Galaxy 3, complete the stage, and fight the Galaxy 3 boss **The Zenith Core Sentinel**.

## In Scope
- Add Galaxy 3 as the next stage after the Galaxy 2 boss
- Build the Galaxy 3 environment, presentation, and scrolling megastructure feel
- Implement Galaxy 3 traversal constraints using **restricted-but-reasonable** obstacle navigation
  - Keep obstacle pressure meaningful
  - Do **not** require extremely tight or pixel-perfect tunnel flying for the initial implementation
- Implement Galaxy 3 enemy and encounter content, including:
  - Tracking Drone deployments
  - Coordinated Tier 2 fighter squads of four
  - Fortress-style Tier 3 scrolling structures with turrets / shielded structure behavior as needed for the stage
- Implement the Galaxy 3 boss encounter: **The Zenith Core Sentinel**
- Integrate Galaxy 3 into current rendering/gameplay/audio-visual systems using the existing project style and established patterns from Galaxies 1 and 2
- Keep the **existing item spawn/cycling system** for this implementation rather than introducing a Galaxy 3-specific item rule set

## Design Constraints
- This task should prioritize **functional end-to-end implementation** over final balancing
- If `Galaxy3-ZenithArmadaGrid-Specification.md` and `Project 2043 Specification.md` differ, resolve conflicts **case-by-case** during planning/implementation and call them out explicitly
- Where the Galaxy 3 spec pushes toward highly restrictive navigation, prefer a version that preserves the intended feel while remaining practical and fun in the current game

## Acceptance Criteria
- Galaxy 3 is reachable in normal progression after defeating the Galaxy 2 boss
- Galaxy 3 has a complete playable flow from stage start through boss defeat
- The stage includes Galaxy-appropriate environment hazards / traversal pressure, enemy waves, fortress encounters, and boss encounter behavior
- The Zenith Core Sentinel is implemented and can be defeated in a normal run
- Existing core systems continue to work through Galaxy 3 integration, including progression, combat, collisions, rendering, and item cycling behavior
- Automated tests are added or updated where feasible
- Manual verification is performed by starting the game from the beginning and playing forward through the existing progression into Galaxy 3
- The project builds successfully after the changes

## Out of Scope
- Full difficulty, pacing, and balance tuning pass
- Spec-accurate ultra-tight or pixel-perfect tunnel navigation
- Major new art, audio, or shader polish beyond what is needed to make Galaxy 3 fit the current game
- Broad refactors or redesign of earlier galaxies
- Progression/save/UX work beyond wiring Galaxy 3 after the Galaxy 2 boss
- Post-Galaxy-3 content, alternate modes, or follow-on stage expansion
